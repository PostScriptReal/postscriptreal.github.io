Friday Night Funkin' Modding ++

FNF Modding ++ is a fork of BulbyVR's FNF Modding+ which was a modding engine for FNF that would allow users to make mods without needing access to source code, making the process easier and more efficient.

Modding++ was created after Modding+ had been abandoned in late November of 2021 (before the last commit in the source code), I loved the engine and didn't want to switch to another one that was still being actively maintained so I forked it and made improvements of my own. This resulted in the birth of the project.

Eventually, after almost a year of development, I abandoned Modding++ because I stopped playing FNF and didn't want to continue a project that wasn't going to end up being released. As such it has been archived on my Github and left in an incomplete, yet playable/usable state.

Post Mortem Patch

An updated version is currently in the works that plans to fix issues and bugs in the last build and give the project a final version that would be much more satisfying and complete, no massive new features are planned to be added and the patch is being worked on seperately. The archived, incomplete version will stay for preservation purposes.

Source code and Build Downloads

If for whatever reason you'd like to mess around with M++, there is a link to the github archive page of the project that hosts old prototype builds, source code and the last build before abandonment. Note that the source code for the old builds are lost.
